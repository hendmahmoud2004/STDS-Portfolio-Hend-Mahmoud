# Task Description

## Project Title
Sales Data Analysis and Visualization using Tableau

## Objective
The objective of this task is to analyze a sales dataset and transform raw data into meaningful insights using interactive dashboards. The project focuses on understanding sales performance, identifying trends, comparing regions, and evaluating product performance to support data-driven decision-making.

## Requirements
- Analyze sales data from a structured dataset.
- Build interactive dashboards using Tableau.
- Visualize key performance indicators (KPIs) such as total sales, trends over time, and top-performing products.
- Enable filtering and comparison across regions and product categories.
- Present insights in a clear and visually appealing way.

## Expected Outcome
- Three dashboards that provide different analytical perspectives:
  1. Sales Overview
  2. Regional Sales Analysis
  3. Product Performance Analysis
- Clear visual insights that help stakeholders understand sales behavior and performance.
