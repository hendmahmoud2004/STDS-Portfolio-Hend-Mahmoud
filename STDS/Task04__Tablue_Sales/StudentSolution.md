# Student Solution

## Overview
This project was implemented using Tableau to analyze sales data through multiple interactive dashboards. Each dashboard focuses on a specific analytical aspect to provide comprehensive insights into the dataset.

---

## Dashboard 1: Sales Overview
### Purpose
To provide a high-level summary of overall sales performance.

### Key Insights
- Total sales and overall performance metrics.
- Monthly sales trends to identify seasonality.
- Comparison between different product categories.
- Identification of top-selling products.
### Dashboard Preview
![Sales Overview Dashboard](sales_overview_dashboard.png)

---

## Dashboard 2: Regional Sales Analysis
### Purpose
To analyze sales distribution and performance across different regions.

### Key Insights
- Sales comparison between regions.
- Regional contribution to total revenue.
- Trend analysis per region.
- Interactive filtering by region and category.
### Dashboard Preview
![Regional Sales Dashboard](regional_sales_dashboard.png)
---

## Dashboard 3: Product Performance Analysis
### Purpose
To evaluate individual product performance and identify strengths and weaknesses.

### Key Insights
- Top-performing and low-performing products.
- Sales and revenue by product and category.
- Product contribution to overall sales.
- Drill-down analysis for detailed exploration.
### Dashboard Preview
![Product Performance Dashboard](product_performance_dashboard.png)

---

## Tools Used
- Tableau Desktop / Tableau Public
- CSV / Excel dataset as the data source
