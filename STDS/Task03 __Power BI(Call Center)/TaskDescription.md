# Task Description: Call Center Data Analysis using Power BI

## Objective
The objective of this task is to analyze Call Center operational data using Power BI
in order to evaluate performance, service efficiency, and customer experience.

---

## Task Requirements
- Import and clean the Call Center dataset
- Build a data model with correct relationships
- Create a dedicated Date Table for time-based analysis
- Calculate key performance indicators (KPIs) using DAX
- Design an interactive Power BI dashboard
- Provide insights and interpretations based on the analysis

---

## Key Metrics to Analyze
- Total number of calls
- Total and average call duration
- SLA compliance (Response Time)
- Call distribution by channel
- Call distribution by reason
- Customer sentiment analysis
- Performance by call center and location

---

## Tools & Technologies
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

---

## Expected Outcome
An interactive dashboard that supports decision-making and clearly presents
Call Center performance insights.
